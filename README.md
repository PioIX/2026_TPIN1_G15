# Descripción

**Who Are Ya? (Futboldle)** es un juego inspirado en Futboldle, donde el jugador debe adivinar un futbolista seleccionado aleatoriamente por el sistema.

La base de datos contiene jugadores de las principales ligas europeas:

- La Liga
- Premier League
- Serie A
- Ligue 1

El usuario deberá escribir nombres de futbolistas hasta descubrir cuál fue el elegido.

---

# Funcionamiento

Cada intento devuelve información sobre distintas características del jugador.

Las categorías son:

- Nacionalidad
- Liga
- Club
- Posición
- Número de camiseta
- Edad

Cada categoría puede mostrarse de tres maneras:

🟢 Correcto

🔴 Incorrecto

⬆️ / ⬇️ Mayor o menor (edad y dorsal)

Las imágenes de banderas, ligas y escudos permiten una experiencia visual más intuitiva.
